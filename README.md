## Installation
To run this project, install it locally using git:

1.- The first step is to clone the repository using the following command
```
$ https://github.com/christian-hume/golang-gorillamux-microservice-restful.git
```

2.- Open project with Visual Studio Code
```
/golang-gorillamux-microservice-restful
```

3.- Run project with Visual Studio Code
```
Run go run main.go handlers.go
```

3.- To test if the project is running, you should access the following URL in a web browser
```
http://localhost:8080/golang/api/v1/users
```
